## gitコマンドの使い方

### ブランチ作成
- `git branch -a` ブランチ確認(`-a`)
- `git branch #` ブランチ作成(`#`=ブランチ名)
- `git checkout -b #` ブランチ移動(`-b` オプション=新規作成)

### コミット
- `git status` ファイルの編集状態確認
- `git add #` 追加したいディレクトリ=`#`
- `git commit -m "#"` addしたディレクトリをコミットする。コメント=`#`

- `git push #1 #2` コミットした結果をプッシュする。`#1`=コミット, `#2`=ブランチ名
